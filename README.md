# G-LAB-3-Creating-Personal-Access-Token-in-GitHub

# Objective:
To generate a Personal Access Token (PAT) in GitHub for authentication.
# Requirements:
GitHub account
Internet access

# Procedure:
Open GitHub Settings:
Login to GitHub
- Click on profile icon
- Go to Settings
- Navigate to Developer Settings:
- Scroll down and click on Developer Settings
- Open Personal Access Tokens:
- Click on Personal Access Tokens
- Select Tokens (classic)
- Generate New Token:
- Click on Generate new token (classic)
- Select required permissions (e.g., repo)
- Create Token:
- Click Generate Token
- Copy and Save Token:
  
Copy the token immediately
Store it safely (it won’t be shown again)
# Result:
Personal Access Token successfully generated for secure GitHub authentication.
# Conclusion:
This lab helps in understanding GitHub authentication using tokens instead of passwords for secure operations.
